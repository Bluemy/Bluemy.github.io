Process

Here are the process of making this portfolio website.

Steps Colour palette Typologies Reference

Step 1: Planning

Before I started creating the website, I went over the examples and components from https://getbootstrap.com to get some inspiration to make my own website. I hope that the overall look of this website is clean so that vistitors can clearly see the content I presented. To achieve that, I planned to use light color as the main color theme.

Step 2: Create the navbar and footer for the pages.

In this step, I have started building the website. The first thing I do is creating the navbar and footer for each pages, since these two element will remain same for each cite. I can get an overview of the cite and do the next step.

For the navbar, the oringinal one I designed was set to be "fixed-top". However, when I scale down the website, the text in the banner part will be disappear(automatically going up). So I delete that setting.

For the footer, I decided divide it into three part, "social media accounts with icons", "my contact infomation" and "sign up form". (I found "sign up form" and "search bar" seems useless for a personal website. But I always seach things in a website, that's the quick way to find things, so for utility reason, I created these two forms. )

Step 3: Create the homepage and inner pages.

In this step, I have selected distinct typologies for each pages. I use Bootstrap framework and dynamic visual components.

Colour palettes

The main colour I use for this website is white and blue. I use the website http://tools.jb51.net/color/color_degress/184,204,214 to find safety colour and contrast colour to match the colour I choose.


For the button, I decided to use light-pink as the colour for hover to match the light-blue, so that visitors could see the difference when click the button.

Typologies

For each page, I have selected distinct typologies.

For the homepage, I want it to be attractive, so I use mixture of photo and text with preview elements. I have included cards and carousel for it.

For the gallery, I decided to separate it into two parts, one is "photo-rich" page with a focus on images and another is "rich-media" page with a focus on videos.

For the resume, I set it into a tabular page with a focus on diagram.

For the contact, I set it into a form page with a focus on forms.

I selected "content-heavy" page as the one for "process page", with a mixed text and images.
